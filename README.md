# TheHayEscape

I made this game as a final project for my JavaScript Game Development class. This is a survivor like where you take the role of a hay bale escaping from a farm whilst being chased by hungry cows. To play, onoline, you can head to x. To download, you can get the zip, extract it, and run it in your preferred browser. Cannot guarantee it will work on all browsers, but it works on Brave, and Chrome to my knowledge. Enjoy!
